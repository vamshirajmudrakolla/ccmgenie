SmartCOMM Demo Portal​


What is the purpose:

This is a php program that helps Integration of multiple components i.e. Draft Editor from Smartcommunication, AWS Email client and Digital signature with Docusign, through a common user interface (UI). ​ ​
This program can be reused with minimal changes in configuration. The inline documentation within the program is self explanatory and can guide the developer use the code effectively.​ ​

We have developed the code based on use cases from our experiences in various engagements where the integration consumed considerable amount of effort in development and testing. It also helps standardize the code base with proven practices across project teams and across engagements.​

Reuse of this code helps a saving of development and testing time. 


Workflow Diagram:​

  • How to effectively use the API services of various platforms such as Draft Editor, Digital signature, Email servers etc.
  • Standardization of best practices in integrations
  • Effort savings
  • We can develop POCs for multiple use cases – Faster to market



This can be demoed by taking the audience through the work-flow and give them a perspective of the integrations between various platforms – it can be followed by a code walk through as a deep dive session ​


The Demo time is 30 min to go over the kit and all the flow.​​

Detailed discussions on specific APIs would take further time.
