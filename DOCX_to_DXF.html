PDF/DOCX to DXF Converter

  • Banking/Insurance industry is going through major IT transformation that involves huge volume of document format creation or migration (from existing legacy systems).
  • This utility converts Microsoft Word Format (DOCX) or PDF to Exstream Dialogue Exchange Format (DXF).  Utility can be used to convert one document at a time or can run a batch with multiple documents. Utility produces a statistics of the passed/ failed cases along with the DXF output. 
  • This utility reduces the migration effort in designing the forms manually from the mockups copying the verbiage/content one by one. 
  • This utility can be configured to handle any version of Exstream by simply updating the XSL file provided by OpenText Exstream.   ​




  • Enable bulk conversion from Docx and PDF to Dialogue Exchange Format (DXF) and importing into OpenText Exstream database. This reduces the designing effort by nearly 50% and helps greatly increase speed of execution
    • ​
  • Rather than designing from scratch, manual effort is limited to pre-cleaning and unit testing of templates, avoid human errors occurred while copying the content from the docx or pdf sample or typing errors thus improve the quality.​

  • Desktop version of MS Word
    • Checkboxes, Radio Buttons, Superscripts and Subscripts, Special characters, Images, Table Borders are not captured in DXF during this conversion
    • The Demo time is 15 min to go over the utility and the overall flow
    • Launch the application and specify the input (source files to be converted) and output (converted output file destination) directories, on click of convert all the PDF files in the input are converted to DOCX and then all the DOCX files are converted to DXF. Complete list of converted files are shown in the application in a grid.
    • To match the Design manager version with the converter, replace the dxfconverter.xsl file in the xsl folder of the converter with the dxfconverter.xsl found in your OpenText installed folder (…\OpenText\Exstream\Exstream 16.6.0\)
