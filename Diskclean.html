SmartCOMM Appliance Disk clean-up utility​


In a typical SmartCOMM development project, the lower environments are configured with limited Disk space that runs out frequently (at least two/ three times a week). The project team need to clean the Disk spaces, manually, by going into folders of Appliances.This task takes about 30-45 min to get completed and the project team must wait until it is done. ​ ​

In order to automate this process and make it run in the background, ValueMomentum has developed a utility using Java. User can configure this utility in any SmartCOMM environment to set a) the frequency (daily, weekly, monthly etc.,) of run and b) a Deletion policy that is based on the age of the files. This utility connects to an Appliance and deletes job files in a server by using directory and Deletion policy parameters. ​

Once configured in an environment, this utility automatically takes care of cleaning the disk that saves precious time for developers. The configuration is done with help of an easy to use UI.
