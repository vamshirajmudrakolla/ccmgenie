SmartCOMM Appliance Restart Utility​

In a typical SmartCOMM development project, the lower environments are configured with Appliance which will manage to run our jobs frequently. If one job is in the state of “In Progress” the remaining jobs which was submitted next to that job will go to “Waiting” state, until unless the “In Progress” job is completed or stopped. To do that manually we need to open the SmartCOMM Appliance, check the jobs and "Stop" that job which is "In Progress" for a longer time period than usual causing blocker for queued jobs to execute.​ ​

Post this, we should Pause and Resume the Appliance so that next jobs will continue to execute.

In order to automate this process, ValueMomentum has developed a utility using PHP. User can configure this utility for any SmartCOMM Appliance to stop the job if it did not respond for longer time and will resume the Appliance to process the waiting jobs. ​

This utility UI is simple to configure tenancy details, Appliances, Job wait policy. This utility monitors the Appliance using the configuration details, we can start/stop this utility from the same UI. It also gives an option to view logs (list of stopped job details).​​
